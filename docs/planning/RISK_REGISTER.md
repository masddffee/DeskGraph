Scale: likelihood and impact are `low`, `medium`, or `high`. A risk remains open until evidence closes it.

| R-004 | Watcher event storms create duplicates or process unstable downloads        | High       | High   | ADR-016 durable per-scope debounce, bounded snapshot/open-handle stability gate, atomic identity-based rescan and restart/rename fixtures pass. Exact `notify 8.2.0` feeds a 256-item non-blocking queue; at most 64 signals drain per cycle, while watch-set changes, overflow, rescan, source failure and unmatched routing force all-scope reconciliation. Multiple distinct paths for one scope request per-scope recovery rather than dropping later rename paths. The original event age caps the wake and forces root-only metadata reconciliation at five minutes, so sustained writes cannot starve a scope. A recovery signal during an old multi-batch scan persists until a fresh post-signal root scan starts. Forced work transactionally binds the exact stored authorization/completed scan; direct ignored observations cannot cancel unrelated stabilizing work, explicit transitions merge into bounded operational aggregates, and hint/scanner paths share temp-suffix exclusion. macOS arm64 live create/modify/rename/delete previously passed outside the restricted sandbox; deterministic temp→final fixtures pass but the expanded live host recheck is pending after external execution quota denial. Windows/Linux/macOS Intel runtime, real kernel overflow/root replacement and load storms remain | M6 | Mitigating |
| R-005 | OCR/image/Office/PDF input causes resource exhaustion or code execution     | Medium     | High   | Controlled-source text/PDF/Office/image boundaries pass. ADR-024's path-free macOS and Windows PNG/JPEG providers enforce source/dimension/pixel/output/observation/chunk/deadline limits, exact pre-publication byte reread, complete-only spatial provenance, and one process-wide native execution slot with bounded first-waiter reservation. The macOS evidence runner adds strict private manifest, digest/identity, 32 MiB case, 512 MiB corpus and one-hour run caps while reusing production image/OCR validation; held directory descriptors, `O_NOFOLLOW_ANY`, relative `openat`, output-parent identity revalidation and `RENAME_EXCL` fixtures fail closed on asset symlinks and directory replacement. Windows additionally owns bounded bytes, rejects rotated/non-finite source coordinates, stores absent confidence honestly, waits for terminal status before close, bounds caller return, and transfers the slot to at most one draining worker. Host policy/state-machine and runner adversarial tests pass; broader concurrent stress/native platform tests, actual Windows/MSIX OCR/cancel/cleanup, representative corpus, Intel/fallback runtime and 8 GB evidence remain | M2          | Mitigating |
