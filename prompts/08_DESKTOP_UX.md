# Phase 08 — Desktop Product UX

Implement milestone M8.

Screens:
1. Onboarding and scope selection
2. First scan and immediate findings
3. Home dashboard
4. Search
5. Projects
6. Project detail
7. Smart Inbox
8. Action preview
9. History and Undo
10. Settings
11. Models and storage
12. Privacy and diagnostics

UX principles:
- explain value, not graph terminology
- show useful findings before deep indexing finishes
- surface privacy and current scan scope
- provide obvious pause
- never use dark patterns to enable auto organization
- show confidence in human language
- every empty and error state needs a next action
- support keyboard navigation and screen readers

Acceptance:
- complete onboarding-to-search E2E test
- complete preview-to-undo E2E test
- responsive at minimum supported sizes
- no raw stack traces in user UI
