# Phase 15 — Launch and Marketing

Use `07_LAUNCH_MARKETING_PLAN.md`.

Create final, channel-specific assets:
- Show HN post
- X thread
- LinkedIn post
- Product Hunt listing and maker comment
- Reddit variants for relevant communities
- Dev.to technical article
- YouTube title, description and chapters
- launch email / direct outreach note
- press kit
- 30-day content calendar

Rules:
- do not ask for fake votes, coordinated upvotes or star exchanges
- be explicit about beta limitations
- link directly to something users can download and run
- reply to technical feedback with evidence
- prioritize installer and data-safety issues

If social credentials are available through approved tools, publish only after the GitHub release is publicly verified. Otherwise output ready-to-post copy and a posting checklist.

Create a launch dashboard template for:
- repo visitors
- stars
- release downloads
- activation
- first scan completion
- search success
- issues and crash-free sessions
