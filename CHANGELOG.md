# Changelog

All notable changes to DeskGraph will be documented here. The project follows Semantic Versioning once release tags begin.

## [Unreleased]

### Changed

- Accepted ADR-033 and implemented bounded local slices: one native selection may atomically authorize 1–32 non-overlapping roots, each with its own opaque platform grant and explicit later metadata scan; the CLI `scope add` path now atomically records a versioned path-free current-host consent receipt so its public add → scan → search flow uses the same active-grant query contract. Settings can add a native-picker hard exclusion after revalidating active/live/host scope policy, canonical stable identity/kind and policy revision. Applying it advances the revision and atomically purges affected derived manifest, FTS, content/OCR, image-metadata, graph/project/relation/screenshot/cleanup and pristine Preview-only action data without changing a source file; receipts are path-free. Excluded stable identities are durable, so an unscanned hard-link alias cannot republish the same bytes through rescan, extraction or search. Vector embeddings are not implemented yet and must join the same policy-revision/purge contract before they ship. Active-grant schema guards cover every writer, and upgrades fail closed by quarantining narrower legacy grants when a broad/narrow pair already exists. Exclusion removal, signed macOS/packaged Windows, clean-machine, hostile-process, cross-platform race/stress and crash evidence remain release work.
